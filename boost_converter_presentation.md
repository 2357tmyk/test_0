## スライド 3: 伝達関数とは
### システムの入出力関係を数学的に表現

```
G(s) = Y(s)/X(s) = K × (分子多項式)/(分母多項式)
```

### 具体例：RC回路
```
Vin ──R─── Vout
      │
      C
      │
     GND
```

**伝達関数：** G(s) = 1/(RCs + 1)

**物理的意味：** 入力信号がシステムを通過する際の変化を表現

---

## スライド 4: 極と零の定義
### システムの特性を決定する重要な概念

**極（Poles）：** 分母 = 0 となるs平面上の点
- システムの固有特性を表現
- 過渡応答の形状を決定

**零（Zeros）：** 分子 = 0 となるs平面上の点  
- 入力から出力への伝達特性を変化
- 周波数応答の形状に影響

### 例：G(s) = (s + 5)/((s + 1)(s + 2))
- 零：s = -5
- 極：s = -1, s = -2

---

## スライド 5: 支配極の概念
### システム応答に最も影響する極

**支配極：** 虚軸に最も近い極
- 最も遅い時定数を持つ
- 過渡応答の主要成分を決定

### 例：極が s₁ = -1, s₂ = -10 の場合
- |Re(s₁)| = 1 < |Re(s₂)| = 10
- **s₁ = -1 が支配極**
- 時定数：τ = 1秒（整定時間 ≈ 4秒）

**設計指針：** 支配極により応答速度が決まる

---

## スライド 6: 極の位置と過渡応答
### s平面での極配置と時間応答の関係

**実極 s = -a の場合：**
- 時間応答：e^(-at)
- 時定数：τ = 1/a
- a が大きい → 速い応答

**複素極 s = -σ ± jωd の場合：**
- 減衰振動応答
- 減衰比：ζ = σ/√(σ² + ωd²)
- ζ < 1：振動的、ζ ≥ 1：非振動的

### 設計のポイント
速い応答 ⇔ 左半平面で虚軸から離れた位置に極配置

---

## スライド 7: ボード線図の基礎
### 周波数応答を視覚化するツール

**2つのプロットで構成：**
1. **振幅プロット：** 20log₁₀|G(jω)| [dB] vs log(ω)
2. **位相プロット：** ∠G(jω) [度] vs log(ω)

### 基本要素の特性
- **定数K：** 振幅：20log₁₀(K) dB、位相：0°
- **積分器1/s：** 振幅：-20dB/decade、位相：-90°
- **微分器s：** 振幅：+20dB/decade、位相：+90°

**利点：** 複雑なシステムも基本要素の重ね合わせで解析

---

## スライド 8: 1次極の周波数応答
### G(s) = 1/(τs + 1) の特性

**コーナー周波数：** ωc = 1/τ

### 周波数領域の特徴
- **低周波（ω << ωc）：** ゲイン≈0dB、位相≈0°
- **コーナー周波数（ω = ωc）：** ゲイン=-3dB、位相=-45°
- **高周波（ω >> ωc）：** ゲイン=-20dB/decade、位相≈-90°

### 設計例：τ = 0.1秒のRC回路
- ωc = 10 rad/s（約1.6 Hz）
- 10 Hz で約-20dB の減衰

---

## スライド 9: 2次系の周波数応答
### G(s) = ωn²/(s² + 2ζωns + ωn²) の特性

**パラメータ：**
- 自然周波数：ωn
- 減衰比：ζ

### 減衰比による応答変化
- **ζ < 0.707：** 共振ピーク発生
- **ζ = 0.707：** 最も平坦な応答
- **ζ > 1：** なだらかな応答

**共振ピーク：** Mr = 1/(2ζ√(1-ζ²))

**制御設計への応用：** ζ = 0.707 付近で設計することが多い

---

## スライド 10: フィードバック制御系
### 基本構成と特徴

```
R(s) ─(+)─ C(s) ─ G(s) ─ Y(s)
      │                    │
      └──── H(s) ──────────┘
           (-)
```

**開ループ伝達関数：** L(s) = C(s)G(s)H(s)
**閉ループ伝達関数：** T(s) = C(s)G(s)/(1 + C(s)G(s)H(s))

### フィードバックの効果
- 定常偏差の改善
- 外乱抑制
- パラメータ変動に対するロバスト性

---

## スライド 11: 安定性余裕
### システムの安定性を定量評価

**ゲイン余裕（GM）：**
- 位相-180°での余分なゲイン
- GM = -20log₁₀|L(jωpc)| [dB]
- 目安：GM > 6 dB

**位相余裕（PM）：**
- ゲイン0dBでの余分な位相
- PM = 180° + ∠L(jωgc) [度]
- 目安：PM > 45°

**意味：** 安定性の「のりしろ」を表す指標

---

## スライド 12: ボード線図による安定性評価
### 視覚的な安定性判定

**安定系の特徴：**
- ゲイン曲線が0dBを横切る前に、位相が-180°に達していない
- GM > 0, PM > 0

**不安定系の特徴：**
- 位相が-180°を下回っている周波数でゲインが正
- GM < 0 または PM < 0

### 実践的な設計指針
1. 充分な位相・ゲイン余裕の確保
2. ゲインクロスオーバー周波数での位相確認
3. 高周波での十分なゲイン減衰

---

## スライド 13: ラウスの判定法
### 特性多項式の係数から安定性を判定

**特性多項式：** D(s) = ans^n + an-1s^n-1 + ... + a1s + a0

**ラウス表の作成：**
```
s³ |  a3   a1
s² |  a2   a0  
s¹ | (a2×a1-a3×a0)/a2  0
s⁰ |  a0   0
```

**安定条件：** 第1列の符号がすべて正

### 設計例：K の安定範囲
D(s) = s³ + 4s² + 5s + K = 0
**結果：** 0 < K < 20 で安定

---

## スライド 14: ナイキスト判定法
### 開ループ特性から閉ループ安定性を評価

**ナイキスト線図：** L(jω) を複素平面にプロット（ω：0→∞）

**判定条件（最小位相系）：**
ナイキスト線図が点(-1, 0)を囲まなければ安定

### 利点
- 視覚的で直感的
- ゲイン・位相余裕が同時に読み取れる
- 非線形要素の影響も扱える

**応用：** 制御器設計時の安定性確認ツールとして活用

---

## スライド 15: PI制御の基礎
### 比例項 + 積分項による制御

**PI制御器：** C(s) = Kp + Ki/s = Kp(1 + 1/(Tis))

**各項の役割：**
- **P項（比例）：** 現在の偏差に応答、速い制御
- **I項（積分）：** 偏差の蓄積に応答、定常偏差除去

### 特徴
✅ 定常偏差を零にできる
❌ 位相遅れにより安定性に影響

**設計のポイント：** 応答性と安定性のバランス

---

## スライド 16: PI制御の周波数特性
### 周波数領域での理解

```
|C(jω)| = Kp√(1 + (1/(ωTi))²)
∠C(jω) = -tan⁻¹(1/(ωTi))
```

**特性：**
- **低周波：** 積分特性（ゲイン∝1/ω、位相-90°）
- **コーナー周波数：** ω = 1/Ti で-3dB、-45°
- **高周波：** 比例特性（ゲイン=Kp、位相≈0°）

**設計への示唆：** Ti により低周波特性、Kp により高周波特性を調整

---

## スライド 17: PI制御パラメータ設計法①帯域設計
### 目標帯域と位相余裕を指定した設計

**設計手順：**
1. 目標のゲインクロスオーバー周波数 ωgc を設定
2. 目標の位相余裕 PM_target を設定
3. Ti を決定：位相条件から
4. Kp を決定：ゲイン条件から

### 設計例
プラント：G(s) = 10/(s(s+1))、目標：ωgc=2rad/s、PM=45°

**結果：** Kp = 0.4、Ti = 0.16s

**利点：** 系統的で設計仕様に直接対応

---

## スライド 18: PI制御パラメータ設計法②極零相殺
### プラントの極をPI制御器の零で相殺

**設計原理：**
プラント：G(s) = K/(s(Ts+1))
PI制御器：C(s) = Kp(1 + 1/(Tis))

**相殺条件：** Ti = T

**結果：** L(s) = KpK/s² （純積分系）

### 特徴
✅ 設計が簡単
✅ プラント極の影響除去
❌ 完全相殺は実際には困難
❌ モデル誤差に敏感

## スライド 19: 状態空間モデルとは
### システムの内部状態を明示的に扱う表現

**状態方程式：**
```
ẋ(t) = Ax(t) + Bu(t)  （状態方程式）
y(t) = Cx(t) + Du(t)  （出力方程式）
```

**利点：**
- MIMO系の統一的取扱い
- 内部状態の直接制御
- 現代制御理論の基礎

### 状態変数の選び方
**物理系：** エネルギー蓄積要素の状態
- 電気系：コンデンサ電圧、インダクタ電流
- 機械系：位置、速度

---

## スライド 20: RLC回路の状態空間モデル例
### 具体的なモデル導出過程

**回路：** 直列RLC回路
**状態変数：** x₁ = iL、x₂ = vC

**状態方程式：**
```
[diL/dt] = [-R/L  -1/L] [iL] + [1/L] vin
[dvC/dt]   [1/C    0  ] [vC]   [0  ]

vout = [0  1] [iL]
              [vC]
```

**物理的意味：**
- 状態の時間変化 = システム内部の相互作用 + 外部入力
- 出力 = 状態の線形結合

---

## スライド 21: 昇圧コンバータの動作原理
### 入力電圧より高い出力電圧を生成

```
Vin ─L─ D ─ Cout ─ Rload
    │      │   │
    │      │   Vout
    SW     │   
    │      │   
   GND ────┘   
```

**動作モード：**
1. **SW ON：** インダクタにエネルギー蓄積
2. **SW OFF：** 蓄積エネルギーを出力に転送

**理想電圧比：** Vout/Vin = 1/(1-D)

D：デューティ比、Vin：入力電圧、Vout：出力電圧

---

## スライド 22: 昇圧コンバータの状態方程式
### スイッチ状態に応じた状態方程式

**状態変数：** iL（インダクタ電流）、vout（出力電圧）

**SW ON時：**
```
[diL/dt  ] = [0   0   ] [iL  ] + [1/L] vin
[dvout/dt]   [0  -1/RC] [vout]   [0  ]
```

**SW OFF時：**
```
[diL/dt  ] = [0   -1/L ] [iL  ] + [1/L] vin  
[dvout/dt]   [1/C -1/RC] [vout]   [0  ]
```

**時間重み付け平均化：** デューティ比dで重み付け平均

---

## スライド 23: 平均化状態方程式
### 連続時間での統一表現

**平均化モデル：**
```
[diL/dt  ] = [0        -(1-d)/L] [iL  ] + [1/L] vin
[dvout/dt]   [(1-d)/C  -1/RC   ] [vout]   [0  ]
```

**平衡点（定常値）：**
- IL = Iload/(1-D)
- Vout = Vin/(1-D)

**意義：**
- スイッチングを平滑化した連続モデル
- 制御設計に適用可能
- 物理的直感と一致

---

## スライド 24: 小信号解析
### 平衡点周りの線形化

**摂動の定義：**
```
iL = IL + îL     （IL：平衡点、îL：摂動）
vout = Vout + v̂out
d = D + d̂
vin = Vin + v̂in
```

**小信号状態方程式：**
```
[dîL/dt  ] = A[îL  ] + B₁v̂in + B₂d̂
[dv̂out/dt]    [v̂out]
```

**意義：** 大信号の非線形モデルから小信号の線形モデルを導出

---

## スライド 25: 主要伝達関数①：G_vd(s)
### デューティ摂動から出力電圧への伝達関数

```
G_vd(s) = v̂out(s)/d̂(s)
```

**導出結果：**
```
G_vd(s) = Vout×(1-D)/(LC) × (1 + s×RC/(1-D)) / [s² + s/(RC) + (1-D)²/(LC)]
```

**特徴：**
- DCゲイン：Vout/(1-D) = Vin/(1-D)²
- 右半平面零の存在（D > 0.5で）
- 2次システムの分母（固有特性）

**制御への影響：** RHP零により帯域制限が必要

---

## スライド 26: 主要伝達関数②：G_id(s)
### デューティ摂動からインダクタ電流への伝達関数

```
G_id(s) = îL(s)/d̂(s)
```

**導出結果：**
```
G_id(s) = [Vout/L × (s + (1-D)²ωn)] / [s² + 2ζωns + ωn²]
```

**特徴：**
- 電流制御ループ設計に使用
- 左半平面零（安定な零）
- 電流の直接制御が可能

**応用：** 内側の電流ループ制御器設計

## スライド 27: その他の伝達関数
### 完全な小信号モデル

**入力電圧摂動関連：**
- G_vg(s)：v̂in → v̂out（電源変動抑制）
- G_ig(s)：v̂in → îL（電源変動の電流への影響）

**負荷電流摂動関連：**
- G_vi(s)：îload → v̂out（負荷変動に対する電圧変化）
- G_ii(s)：îload → îL（負荷変動の電流への影響）

**総合的な制御設計：** 全ての外乱・変動を考慮した制御系設計が可能
