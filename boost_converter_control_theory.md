## 1. 伝達関数

### 1.1 伝達関数の基礎

伝達関数は、線形時不変システムの入力と出力の関係を、ラプラス変換の比として表現したものです。システムG(s)の伝達関数は以下のように定義されます：

```
G(s) = Y(s)/X(s)
```

ここで、Y(s)は出力のラプラス変換、X(s)は入力のラプラス変換、sは複素変数です。

一般的に、伝達関数は以下のような有理関数として表現されます：

```
G(s) = K × (b_m×s^m + b_{m-1}×s^{m-1} + ... + b_1×s + b_0) / (a_n×s^n + a_{n-1}×s^{n-1} + ... + a_1×s + a_0)
```

### 1.2 極と零の定義

#### 極（Poles）
極は、伝達関数の分母を零にするs平面上の点です。つまり、分母多項式の根です。

極は以下のように求められます：
```
a_n×s^n + a_{n-1}×s^{n-1} + ... + a_1×s + a_0 = 0
```

例：G(s) = 10/(s² + 3s + 2) の極
分母 = s² + 3s + 2 = (s + 1)(s + 2) = 0
従って、極は s = -1 と s = -2 です。

#### 零（Zeros）
零は、伝達関数の分子を零にするs平面上の点です。つまり、分子多項式の根です。

零は以下のように求められます：
```
b_m×s^m + b_{m-1}×s^{m-1} + ... + b_1×s + b_0 = 0
```

例：G(s) = (s + 5)/(s² + 3s + 2) の零
分子 = s + 5 = 0
従って、零は s = -5 です。

#### 物理的意味
- **極**: システムの固有特性を表し、安定性や過渡応答の形状を決定します
- **零**: システムの入力から出力への伝達特性を変化させ、周波数応答の形状に影響します

### 1.3 支配極（Dominant Poles）

支配極とは、システムの過渡応答に最も大きな影響を与える極のことです。一般的に、虚軸に最も近い極が支配極となります。

#### 支配極の特定方法
1. すべての極を求める
2. 各極の実部の絶対値を比較
3. 実部の絶対値が最小の極が支配極

例：極が s₁ = -1, s₂ = -10 の場合
|Re(s₁)| = 1 < |Re(s₂)| = 10
したがって、s₁ = -1 が支配極となります。

#### 支配極の重要性
支配極は以下を決定します：
- 整定時間（Settling time）
- 過渡応答の主要な時定数
- システムの安定性マージン

### 1.4 極/零の位置による過渡応答

#### 実極の影響
実極 s = -a (a > 0) は、指数関数的減衰項 e^(-at) を生じます。

- **a が小さい**: ゆっくりとした減衰（長い時定数τ = 1/a）
- **a が大きい**: 速い減衰（短い時定数）

例：
- s = -1 → τ = 1秒、整定時間 ≈ 4秒
- s = -10 → τ = 0.1秒、整定時間 ≈ 0.4秒

#### 複素極の影響
複素極 s = -σ ± jω_d は、減衰振動を生じます：

```
減衰比 ζ = σ/√(σ² + ω_d²)
自然周波数 ω_n = √(σ² + ω_d²)
```

- **ζ < 1**: アンダーダンプ（振動的）
- **ζ = 1**: クリティカルダンプ
- **ζ > 1**: オーバーダンプ

#### 零の影響
零は過渡応答の初期値や波形に影響します：

- **左半平面零（LHP Zero）**: 通常の応答
- **右半平面零（RHP Zero）**: 非最小位相システム、逆応答を生じる

具体例：RHPゼロを持つシステム
```
G(s) = (-s + 1)/(s + 1)²
```
このシステムは初期に逆方向の応答を示した後、目標値に収束します。

## 2. ボード線図

### 2.1 ボード線図の基礎

ボード線図は、伝達関数の周波数応答を表現するツールで、以下の2つのプロットで構成されます：

1. **振幅プロット**: 20log₁₀|G(jω)| [dB] vs log(ω)
2. **位相プロット**: ∠G(jω) [度] vs log(ω)

### 2.2 交流入力信号に対する過渡応答

正弦波入力 x(t) = A sin(ωt) に対するシステムの応答を考えます。

#### 定常状態での応答
線形システムにおいて、正弦波入力に対する定常応答は同じ周波数の正弦波となります：

```
x(t) = A sin(ωt) → y(t) = A|G(jω)| sin(ωt + ∠G(jω))
```

ここで：
- |G(jω)|: 振幅比（ゲイン）
- ∠G(jω): 位相差

#### 物理的解釈
例：RC回路（G(s) = 1/(RCs + 1)）の場合
- 低周波（ω << 1/RC）: ゲイン ≈ 1、位相差 ≈ 0°
- 高周波（ω >> 1/RC）: ゲイン ∝ 1/ω、位相差 ≈ -90°

### 2.3 伝達関数の周波数応答

#### 基本要素の周波数応答

**1. 定数ゲイン K**
```
G(s) = K
振幅: 20log₁₀(K) [dB]（周波数に無関係）
位相: 0° (K > 0), -180° (K < 0)
```

**2. 積分器 1/s**
```
G(s) = 1/s
振幅: -20log₁₀(ω) [dB]（-20dB/decadeで減少）
位相: -90°（周波数に無関係）
```

**3. 微分器 s**
```
G(s) = s  
振幅: 20log₁₀(ω) [dB]（+20dB/decadeで増加）
位相: +90°（周波数に無関係）
```

### 2.4 極を含む伝達関数の周波数応答

#### 1次極 1/(τs + 1)

コーナー周波数：ω_c = 1/τ

```
|G(jω)| = 1/√(1 + (ωτ)²)
∠G(jω) = -tan⁻¹(ωτ)
```

**低周波領域（ω << ω_c）:**
- 振幅: ≈ 0 dB
- 位相: ≈ 0°

**コーナー周波数（ω = ω_c）:**
- 振幅: -3 dB
- 位相: -45°

**高周波領域（ω >> ω_c）:**
- 振幅: -20log₁₀(ω/ω_c) dB（-20dB/decade）
- 位相: ≈ -90°

具体例：τ = 0.1秒のRC回路
- ω_c = 10 rad/s
- 1 Hz（6.28 rad/s）で約-1dB
- 10 Hz（62.8 rad/s）で約-20dB

#### 2次極 ω_n²/(s² + 2ζω_n s + ω_n²)

自然周波数：ω_n、減衰比：ζ

**低周波領域（ω << ω_n）:**
- 振幅: ≈ 0 dB
- 位相: ≈ 0°

**共振周波数近傍（ω ≈ ω_n）:**
- 共振周波数: ω_r = ω_n√(1 - 2ζ²) (ζ < 0.707の場合)
- 共振ピーク: M_r = 1/(2ζ√(1 - ζ²)) (ζ < 0.707の場合)

**高周波領域（ω >> ω_n）:**
- 振幅: -40log₁₀(ω/ω_n) dB（-40dB/decade）
- 位相: ≈ -180°

### 2.5 零を含む伝達関数の周波数応答

#### 1次零 (τs + 1)

コーナー周波数：ω_c = 1/τ

```
|G(jω)| = √(1 + (ωτ)²)
∠G(jω) = tan⁻¹(ωτ)
```

零は極と逆の特性を示します：

**低周波領域（ω << ω_c）:**
- 振幅: ≈ 0 dB
- 位相: ≈ 0°

**高周波領域（ω >> ω_c）:**
- 振幅: +20log₁₀(ω/ω_c) dB（+20dB/decade）
- 位相: ≈ +90°

#### 右半平面零（RHP Zero）

RHP零 (1 - τs) の場合：

```
|G(jω)| = √(1 + (ωτ)²)　（振幅は同じ）
∠G(jω) = -tan⁻¹(ωτ)　（位相は逆）
```

RHP零の特徴：
- 振幅特性は左半平面零と同じ
- 位相特性は逆（負の位相）
- 制御系設計上の問題を引き起こす

## 3. フィードバック制御の安定性

### 3.1 フィードバック制御システム

典型的な単位フィードバック制御系：
```
       R(s) ───(+)─── C(s) ─── G(s) ─── Y(s)
                │                        │
                └──────── H(s) ──────────┘
                       (-)
```

閉ループ伝達関数：
```
T(s) = C(s)G(s)/(1 + C(s)G(s)H(s))
```

開ループ伝達関数：L(s) = C(s)G(s)H(s)

### 3.2 位相余裕（Phase Margin）とゲイン余裕（Gain Margin）

#### ゲイン余裕（GM）
ゲイン余裕は、位相が-180°となる周波数（位相クロスオーバー周波数 ω_pc）でのゲイン余裕です。

```
GM = -20log₁₀|L(jω_pc)| [dB]
```

安定性の目安：GM > 6 dB

#### 位相余裕（PM）
位相余裕は、ゲインが0 dBとなる周波数（ゲインクロスオーバー周波数 ω_gc）での位相余裕です。

```
PM = 180° + ∠L(jω_gc) [度]
```

安定性の目安：PM > 45°

#### 設計例：比例制御系
プラント：G(s) = 1/(s(s+1))
比例制御器：C(s) = Kp

開ループ伝達関数：L(s) = Kp/(s(s+1))

Kp = 1の場合：
- ω_gc = 1 rad/s でゲイン = 0dB
- この時の位相 = -90° - 45° = -135°
- PM = 180° - 135° = 45°

### 3.3 ボード線図との対応

ボード線図から安定性を評価：

1. **安定系の特徴**
   - ゲイン曲線が0dBを横切る前に、位相曲線が-180°に達していない
   - GM > 0 dB, PM > 0°

2. **不安定系の特徴**
   - 位相が-180°を下回っている周波数でゲインが正
   - GM < 0 dB または PM < 0°

3. **限界安定系**
   - GM = 0 dB または PM = 0°

### 3.4 ラウスの判定法

ラウスの判定法は、特性多項式の係数から、ラプラス変換領域で安定性を判定する手法です。

#### 特性多項式
```
D(s) = a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0 = 0
```

#### ラウス表の作成

例：3次システム D(s) = s³ + 4s² + 5s + K = 0

```
s³ |  1    5
s² |  4    K
s¹ | (20-K)/4  0
s⁰ |  K    0
```

#### 安定条件
すべての係数が正で、ラウス表の第1列がすべて正であること。

上記例の安定条件：
- K > 0
- (20-K)/4 > 0 → K < 20

従って、0 < K < 20 で安定。

#### 限界安定の場合
K = 20の時、特性方程式は：
s³ + 4s² + 5s + 20 = 0
限界安定で、純虚数根 s = ±j√5 を持つ。

### 3.5 ナイキスト判定法

ナイキスト判定法は、開ループ伝達関数のナイキスト線図から閉ループ系の安定性を判定する手法です。

#### ナイキストの安定判別法
開ループ伝達関数をL(s)とし、閉ループ系の特性方程式を1 + L(s) = 0とする。

**ナイキストの判別条件：**
ナイキスト線図が点(-1, 0)を反時計回りにP/2回囲むとき、閉ループ系は安定。
ここで、Pは開ループ伝達関数L(s)の右半平面極数。

#### 実用的な判定方法

**最小位相系（P = 0）の場合：**
ナイキスト線図が点(-1, 0)を囲まなければ安定。

**判定手順：**
1. L(jω)を0から∞まで描く
2. 点(-1, 0)との関係を確認
3. ゲイン・位相余裕を読み取り

#### 設計例
L(s) = K/(s(s+1)(s+2))の場合

K = 1時のナイキスト線図：
- ω = 0: L(j0) = ∞∠-90°
- ω = 1: L(j1) ≈ -0.45∠-161°
- ω = ∞: L(j∞) = 0∠-270°

点(-1, 0)を囲まないため安定。
K増加により線図が拡大し、K ≈ 6で限界安定となる。

## 4. PI制御

### 4.1 PI制御概要

PI制御器は比例項（P項）と積分項（I項）で構成される制御器です。

#### PI制御器の伝達関数
```
C(s) = Kp + Ki/s = Kp(1 + 1/(Ti×s))
```

ここで：
- Kp: 比例ゲイン
- Ki: 積分ゲイン  
- Ti = Kp/Ki: 積分時定数

#### P項の役割
比例項は現在の偏差に比例した制御出力を生成します。

**特徴：**
- 応答が速い
- 定常偏差が残る
- ゲインを上げると不安定になりやすい

#### I項の役割
積分項は偏差の累積に応じた制御出力を生成します。

**特徴：**
- 定常偏差を零にできる
- 応答が遅い
- 位相遅れを生じる（安定性に悪影響）

### 4.2 PI制御の周波数特性

PI制御器の周波数応答：
```
C(jω) = Kp(1 + 1/(jωTi))
```

#### 振幅特性
```
|C(jω)| = Kp√(1 + (1/(ωTi))²)
```

- 低周波（ω << 1/Ti）: |C(jω)| ≈ Kp/(ωTi) （積分特性）
- 高周波（ω >> 1/Ti）: |C(jω)| ≈ Kp （比例特性）

#### 位相特性
```
∠C(jω) = -tan⁻¹(1/(ωTi))
```

- 低周波: ≈ -90° （積分による位相遅れ）
- コーナー周波数（ω = 1/Ti）: -45°
- 高周波: ≈ 0° （比例特性）

### 4.3 PI制御パラメータの設計方法

#### 4.3.1 帯域設計法

設計目標：
- 目標のゲインクロスオーバー周波数 ω_gc の設定
- 目標の位相余裕 PM_target の確保

**設計手順：**

1. **プラントの周波数応答を調べる**
   G(s) = K_p/(s(Ts+1)) の例
   
2. **目標帯域での位相余裕を計算**
   PI制御器により位相遅れが生じるため、プラント単体の位相余裕よりも高い値が必要

3. **積分時定数 Ti の設計**
   位相余裕の目標値から：
   ```
   PM_target = 180° + ∠G(jω_gc) - 90° + tan⁻¹(ω_gc Ti)
   ```

4. **比例ゲイン Kp の設計**
   ゲインクロスオーバー周波数での条件から：
   ```
   |Kp(1 + 1/(jω_gc Ti)) G(jω_gc)| = 1
   ```

#### 具体的設計例

プラント：G(s) = 10/(s(s+1))
設計仕様：ω_gc = 2 rad/s, PM = 45°

**Step 1:** ω_gc = 2 でのプラント特性
|G(j2)| = 10/(2√5) = √5/2
∠G(j2) = -90° - tan⁻¹(2) ≈ -153.4°

**Step 2:** 必要な位相余裕（PI制御器による改善を考慮）
目標: 45° = 180° - 153.4° + ∠C(j2)
∠C(j2) = 45° - 26.6° = 18.4°

**Step 3:** Ti の決定
18.4° = -tan⁻¹(1/(2Ti))
Ti = 1/(2×tan(71.6°)) ≈ 0.16 s

**Step 4:** Kp の決定
|Kp(1 + 1/(j2×0.16))| × √5/2 = 1
|Kp√(1 + (1/0.32)²)| = 2/√5
Kp = 0.4

#### 4.3.2 極零相殺法

プラントの主要な極を PI制御器の零で相殺する方法です。

プラント：G(s) = K/(s(Ts+1))
PI制御器：C(s) = Kp(1 + 1/(Ti s))

**設計原理：**
PI制御器の零 s = -1/Ti でプラントの極 s = -1/T を相殺：
```
Ti = T
```

この場合、開ループ伝達関数は：
```
L(s) = C(s)G(s) = Kp K/s²
```

純積分特性となり、設計が簡単になります。

**利点：**
- 設計が簡単
- 相殺された極の影響を除去

**欠点：**
- 完全な相殺は実際には困難
- 相殺されない極の影響で性能劣化の可能性

#### 4.3.3 クロスオーバー条件を用いた設計

ゲインクロスオーバー周波数 ω_gc と位相余裕 PM を指定して設計する方法です。

**設計条件：**
```
|L(jω_gc)| = 1  （ゲイン条件）
∠L(jω_gc) = -180° + PM  （位相条件）
```

**一般的なプラント G(s) = K/(s^n ∏(Tᵢs+1)) の場合：**

1. **位相条件からTiを求める：**
   ```
   PM = 180° - 90°×n - Σ tan⁻¹(ωgc Tᵢ) + tan⁻¹(ωgc Ti)
   ```

2. **ゲイン条件からKpを求める：**
   ```
   Kp = (ωgc^n ∏√(1 + (ωgc Tᵢ)²))/(K√(1 + (ωgc Ti)²))
   ```

この方法は系統的で、多くの実用システムに適用可能です。

## 5. 状態空間モデル

### 5.1 状態空間表現の基礎

状態空間モデルは、システムの内部状態を明示的に扱うことができる表現方法です。

#### 状態空間方程式
```
ẋ(t) = Ax(t) + Bu(t)  （状態方程式）
y(t) = Cx(t) + Du(t)  （出力方程式）
```

ここで：
- x(t): 状態ベクトル（n×1）
- u(t): 入力ベクトル（m×1）
- y(t): 出力ベクトル（p×1）
- A: システム行列（n×n）
- B: 入力行列（n×m）
- C: 出力行列（p×n）
- D: フィードフォワード行列（p×m）

#### 伝達関数との関係
ラプラス変換により：
```
G(s) = C(sI - A)⁻¹B + D
```

### 5.2 状態変数の選び方

状態変数は、システムの将来の挙動を決定するために必要な最小限の情報を表します。

#### 典型的な選択例

**電気回路系：**
- キャパシタの電圧
- インダクタの電流

**機械系：**
- 位置と速度
- 各質量の変位と速度

**制御系：**
- エネルギー蓄積要素の状態

### 5.3 RLC回路の状態空間モデル例

直列RLC回路を例に状態空間モデルを導出します。

#### 回路構成
```
vin ─── R ─── L ─── C ─── GND
                │        
                vout     
```

#### 状態変数の選択
- x₁ = iL （インダクタ電流）
- x₂ = vC （キャパシタ電圧）

#### 状態方程式の導出

**キルヒホッフの電圧法則：**
```
vin = R×iL + L×(diL/dt) + vC
```

**キャパシタの電流電圧関係：**
```
iL = C×(dvC/dt)
```

これらから状態方程式を求めると：
```
d/dt [iL] = [-R/L  -1/L] [iL] + [1/L] vin
     [vC]   [1/C    0  ] [vC]   [0  ]

vout = [0  1] [iL]
              [vC]
```

#### 行列形式
```
A = [-R/L  -1/L]    B = [1/L]    C = [0  1]    D = [0]
    [1/C    0  ]        [0  ]
```

数値例（R=1Ω, L=1H, C=1F）：
```
A = [-1  -1]    B = [1]    C = [0  1]    D = [0]
    [1    0]        [0]
```

### 5.4 可制御性と可観測性

#### 可制御性（Controllability）
すべての状態を入力により任意の値に制御できるかを表します。

**可制御性行列：**
```
Pc = [B  AB  A²B  ...  A^{n-1}B]
```

**可制御性判定：**
rank(Pc) = n なら完全可制御

#### 可観測性（Observability）
すべての状態を出力から推定できるかを表します。

**可観測性行列：**
```
Po = [C]
     [CA]
     [CA²]
     [⋮]
     [CA^{n-1}]
```

**可観測性判定：**
rank(Po) = n なら完全可観測

### 5.5 状態フィードバック制御

状態フィードバック制御では、すべての状態を測定し、それらの線形結合で制御入力を決定します。

#### 制御則
```
u(t) = -Kx(t) + r(t)
```

ここで、K は状態フィードバックゲイン行列、r(t) は目標値です。

#### 閉ループシステム
```
ẋ(t) = (A - BK)x(t) + Br(t)
```

特性方程式：det(sI - (A - BK)) = 0

#### 極配置設計
フィードバックゲイン K を設計して、閉ループ極を所望の位置に配置します。

**設計手順（SISO系）：**
1. 望ましい極 λ₁, λ₂, ..., λₙ を決定
2. 望ましい特性多項式 D(s) = ∏(s - λᵢ) を求める
3. K を求めて det(sI - (A - BK)) = D(s) とする

## 6. 昇圧コンバータの伝達関数導出

### 6.1 昇圧コンバータの基本動作

昇圧コンバータは、入力電圧より高い出力電圧を得る DC-DC コンバータです。

#### 回路構成
```
Vin ───L─── D ─── Cout ─── Rload
       │         │   │
       │         │   Vout
       │         │   
       SW        │
       │         │
       GND ──────┘
```

#### 動作原理

**スイッチON時（0 < t < DT）：**
- インダクタにエネルギー蓄積: Vin = L(diL/dt)
- ダイオードOFF、出力は出力キャパシタから供給

**スイッチOFF時（DT < t < T）：**
- インダクタのエネルギーが出力に転送
- ダイオードON: Vin - Vout = L(diL/dt)

### 6.2 状態空間モデルの導出

#### 状態変数の定義
- iL: インダクタ電流
- vout: 出力電圧

#### スイッチON時のモデル（0 < t < DT）
```
L(diL/dt) = vin
Cout(dvout/dt) = -vout/Rload = -iload
```

状態方程式：
```
[diL/dt  ] = [0    0   ] [iL  ] + [1/L] vin + [0] iload
[dvout/dt]   [0  -1/RC ] [vout]   [0  ]       [-1/C]
```

#### スイッチOFF時のモデル（DT < t < T）
```
L(diL/dt) = vin - vout
Cout(dvout/dt) = iL - vout/Rload
```

状態方程式：
```
[diL/dt  ] = [0   -1/L ] [iL  ] + [1/L] vin + [0] iload
[dvout/dt]   [1/C -1/RC] [vout]   [0  ]       [-1/C]
```

### 6.3 平均化モデルの導出

デューティ比を d として、状態方程式を平均化します。

#### 平均化原理
```
ẋ = d×[ON時の状態方程式] + (1-d)×[OFF時の状態方程式]
```

#### 平均化状態方程式
```
[diL/dt  ] = [0        -(1-d)/L] [iL  ] + [1/L] vin + [0] iload
[dvout/dt]   [(1-d)/C  -1/RC   ] [vout]   [0  ]       [-1/C]
```

行列形式：
```
A(d) = [0        -(1-d)/L]    B₁ = [1/L]    B₂ = [0  ]
       [(1-d)/C  -1/RC   ]          [0  ]          [-1/C]

ẋ = A(d)x + B₁vin + B₂iload
```

### 6.4 平衡点の計算

定常状態では ẋ = 0 なので：
```
0 = A(D)X + B₁Vin + B₂Iload
```

ここで、大文字は定常値を表します。

#### 定常値の解
```
X = -A(D)⁻¹(B₁Vin + B₂Iload)
```

**計算結果：**
```
IL = Iload/(1-D)
Vout = Vin/(1-D)
```

これは昇圧コンバータの理想的な電圧変換比 Vout/Vin = 1/(1-D) と一致します。

### 6.5 小信号モデルの導出

平衡点周りの小信号解析を行います。

#### 摂動の定義
```
iL = IL + îL
vout = Vout + v̂out  
d = D + d̂
vin = Vin + v̂in
iload = Iload + îload
```

#### 摂動状態方程式
線形化により：
```
[dîL/dt  ] = A(D)[îL  ] + ∂A/∂d|D [îL  ]d̂ + B₁v̂in + B₂îload
[dv̂out/dt]        [v̂out]        [v̂out]
```

ここで：
```
∂A/∂d|D = [0      1/L ]
           [-1/C    0  ]
```

#### 摂動項の整理
```
[dîL/dt  ] = [0        -(1-D)/L] [îL  ] + [Vout/L ] d̂ + [1/L] v̂in + [0  ] îload
[dv̂out/dt]   [(1-D)/C  -1/RC   ] [v̂out]   [-IL/C  ]      [0  ]       [-1/C]
```

### 6.6 各伝達関数の導出

#### 6.6.1 G_id(s): dutyの摂動に対するインダクタ電流の応答

入力：d̂、出力：îL

v̂in = 0, îload = 0 とし、îL について解くと：

```
G_id(s) = îL(s)/d̂(s) = [Vout/L × s + (1-D)×Vout/(LC)] / [s² + s/(RC) + (1-D)²/(LC)]
```

分母は特性多項式で、昇圧コンバータの固有モードを表します。

#### 6.6.2 G_vd(s): dutyの摂動に対する出力電圧の応答

入力：d̂、出力：v̂out

```
G_vd(s) = v̂out(s)/d̂(s) = [Vout×(1-D)/(LC) × (1 + s×RC/(1-D))] / [s² + s/(RC) + (1-D)²/(LC)]
```

**特徴：**
- DCゲイン：Vout/(1-D) = Vin/(1-D)²
- 右半平面零：s = -(1-D)/(RC) （D > 0.5で正の値）

#### 6.6.3 その他の伝達関数

**入力電圧の摂動に対する出力電圧の応答：**
```
G_vg(s) = v̂out(s)/v̂in(s) = [(1-D) × (1 + sL/((1-D)²R))] / [s² + s/(RC) + (1-D)²/(LC)]
```

**入力電圧の摂動に対するインダクタ電流の応答：**
```
G_ig(s) = îL(s)/v̂in(s) = [1/L × (1 + s×RC)] / [s² + s/(RC) + (1-D)²/(LC)]
```

**負荷電流の摂動に対する出力電圧の応答：**
```
G_vi(s) = v̂out(s)/îload(s) = [-R × (1 + sL/R)] / [s² + s/(RC) + (1-D)²/(LC)]
```

**負荷電流の摂動に対するインダクタ電流の応答：**
```
G_ii(s) = îL(s)/îload(s) = [1/(1-D) × (1 + s×RC)] / [s² + s/(RC) + (1-D)²/(LC)]
```

### 6.7 数値設計例

#### 設計仕様
```
Vin = 12V
Vout = 24V  
D = 0.5
L = 100μH
C = 100μF
R = 10Ω（負荷抵抗）
fs = 100kHz
```

#### 平衡点
```
IL = Iload/(1-D) = (24/10)/0.5 = 4.8A
Vout = 24V
```

#### 特性周波数
```
自然周波数：ωn = √((1-D)²/(LC)) = √(0.25/(100×10⁻⁶×100×10⁻⁶)) = 1592 rad/s (253 Hz)
減衰比：ζ = 1/(2R)√(L/C) = 1/20×√(100×10⁻⁶/100×10⁻⁶) = 0.05
```

#### 主要伝達関数（数値）

**G_vd(s)（duty → 出力電圧）：**
```
G_vd(s) = 24 × (1 + s×10×100×10⁻⁶/0.5) / (s²/(1592)² + 2×0.05×s/1592 + 1)
        = 24 × (1 + s×2×10⁻³) / ((s/1592)² + s×0.159/1592 + 1)
```

**G_id(s)（duty → インダクタ電流）：**
```  
G_id(s) = (24/100×10⁻⁶) × (s + 0.5×1592) / (s² + 159.2s + 1592²)
        = 240000 × (s + 796) / (s² + 159.2s + 2534464)
```

### 6.8 制御設計への応用

#### 電流ループ設計
内側ループでインダクタ電流を制御：

制御対象：G_id(s)
PI制御器設計により電流フィードバックループを構成

#### 電圧ループ設計  
外側ループで出力電圧を制御：

制御対象：G_vd(s) × 電流ループ閉ループ伝達関数
右半平面零の存在により、帯域制限が必要

#### 安定性の考慮事項
1. **右半平面零の制約**
   - D > 0.5 で G_vd(s) に RHP零が出現
   - クロスオーバー周波数を RHP零の1/3以下に制限

2. **電流連続モードの維持**
   - 最小インダクタ電流 > 0 の条件確保
   - 負荷変動に対する安定動作

3. **補償器設計**
   - 低周波でのゲイン向上（定常偏差除去）
   - 高周波での位相・ゲイン余裕確保

この包括的な解析により、昇圧コンバータの制御システム設計において必要な全ての伝達関数が導出でき、実際の制御器設計に活用できます。
