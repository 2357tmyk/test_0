1. 基礎理論
================================================================================

1.1 パワーエレクトロニクスとは
パワーエレクトロニクスは、電力の効率的な変換・制御を行う技術分野です。主な目的は：
- 電圧変換（昇圧・降圧）
- 交流⇔直流変換
- 電力制御・調整

1.2 基本的な電気回路の法則

◆オームの法則
V = I × R
- V: 電圧 [V]
- I: 電流 [A] 
- R: 抵抗 [Ω]

◆キルヒホッフの法則
電流則（KCL）: 任意の節点で流入電流の和 = 流出電流の和
電圧則（KVL）: 閉回路における電圧の総和 = 0

◆インダクタとキャパシタの基本式
インダクタ（コイル）: V_L = L × (di/dt)
- L: インダクタンス [H]
- di/dt: 電流の時間変化率

キャパシタ（コンデンサ）: I_C = C × (dv/dt)  
- C: 静電容量 [F]
- dv/dt: 電圧の時間変化率

1.3 エネルギー蓄積素子
- インダクタ: 磁気エネルギー蓄積 E_L = (1/2) × L × I²
- キャパシタ: 電気エネルギー蓄積 E_C = (1/2) × C × V²

2. Buck/Boost コンバータの基本
================================================================================

2.1 Buckコンバータ（降圧コンバータ）

◆基本回路構成
Vin ----[SW]----[L]----+---- Vout
         |              |
        [D]           [C]--- R_load
         |              |
        GND -----------GND

◆動作原理
1. スイッチON期間（0 < t < D×T）
   - 電流がインダクタに流れ、エネルギー蓄積
   - ダイオードは逆バイアスでOFF

2. スイッチOFF期間（D×T < t < T）
   - インダクタの蓄積エネルギーが負荷に供給
   - ダイオードがONして電流経路を形成

◆基本設計式
電圧変換比: Vout/Vin = D
- D: デューティ比（0 < D < 1）
- T: スイッチング周期

連続モード時のインダクタ設計: L_min = R × (1-D) / (2×f)
- R: 負荷抵抗
- f: スイッチング周波数

出力リップル電圧: ΔVout = (1-D) / (8×L×C×f²)

2.2 Boostコンバータ（昇圧コンバータ）

◆基本回路構成
Vin ----[L]----+----[D]---- Vout
               |            |
              [SW]        [C]--- R_load
               |            |
              GND ---------GND

◆動作原理
1. スイッチON期間
   - インダクタにエネルギー蓄積
   - 出力はキャパシタから供給

2. スイッチOFF期間
   - インダクタと入力電源が直列接続
   - 昇圧された電圧で出力に供給

◆基本設計式
電圧変換比: Vout/Vin = 1/(1-D)

連続モード時のインダクタ設計: L_min = Vin × D / (ΔI_L × f)
- ΔI_L: インダクタ電流リップル

3. 制御理論の基礎
================================================================================

3.1 フィードバック制御の基本概念

◆制御系の基本構成
指令値 → [制御器] → [制御対象] → 出力
  ↑                               ↓
  +←----- [フィードバック] ←------+

3.2 PI制御器の設計

◆PI制御器の伝達関数
G_PI(s) = Kp + Ki/s = Kp × (1 + 1/(Ti×s))
- Kp: 比例ゲイン
- Ki: 積分ゲイン  
- Ti: 積分時間定数

◆PI制御器の特性
- 比例動作: 偏差に比例した制御出力
- 積分動作: 定常偏差を零に収束
- 位相遅れ: 低周波で90度の位相遅れ

3.3 制御系設計の基本手順
1. 制御対象のモデル化: 伝達関数の導出
2. 仕様設定: 応答時間、オーバーシュート、定常偏差
3. 制御器設計: ゲイン調整、安定性確保
4. 検証: シミュレーション・実機試験

4. 二重PI制御設計手法
================================================================================

4.1 電圧・電流二重ループ制御

◆システム構成
電圧指令 → [電圧PI] → 電流指令 → [電流PI] → PWM → [コンバータ] → 出力電圧
   ↑                      ↑                                        ↓
   +←----------←----------+←------ 電圧フィードバック ←-----------+
                          ↑                                        
                          +←------ 電流フィードバック ←-----------+

4.2 電流制御ループ設計

◆電流制御の目的
- 高速応答の実現
- 過電流保護
- 電圧制御の安定化

◆電流PI制御器設計手順
1. 電流制御対象の伝達関数: G_i(s) = 1/(L×s + R)

2. 電流PI制御器の設計: G_PI_i(s) = Kp_i × (1 + 1/(Ti_i×s))

3. 設計指針:
   - 極零相殺: Ti_i = L/R
   - ゲイン設定: Kp_i = L × ω_ci
   - 電流ループ帯域: ω_ci = fs/10 (fsはスイッチング周波数)

4.3 電圧制御ループ設計

◆電圧制御対象の伝達関数
電流制御ループを1次遅れ系で近似: G_v(s) = 1/(C×s) × 1/(1 + s/ω_ci)

◆電圧PI制御器設計
G_PI_v(s) = Kp_v × (1 + 1/(Ti_v×s))

設計手順:
1. 電圧ループ帯域設定: ω_cv = ω_ci/10
2. 積分時間定数: Ti_v = 1/ω_cv  
3. 比例ゲイン: Kp_v = C × ω_cv²

5. 実設計手順
================================================================================

5.1 Step 1: 仕様設定
- 入力電圧範囲: Vin_min ～ Vin_max
- 出力電圧: Vout
- 最大出力電流: Iout_max
- 効率目標: η
- リップル要求: ΔVout/Vout

5.2 Step 2: 回路パラメータ設計

◆スイッチング周波数の選定
考慮事項:
- 高周波化 → 受動素子小型化、スイッチング損失増大
- 低周波化 → 受動素子大型化、スイッチング損失減少
- 一般的な範囲: 20kHz ～ 200kHz

◆インダクタ設計
Buckコンバータの場合: L = (Vin - Vout) × D / (ΔI_L × f)
- ΔI_L = 0.2 ～ 0.4 × Iout_avg (設計指針)

◆キャパシタ設計
出力キャパシタ: C = ΔI_L / (8 × f × ΔVout)

5.3 Step 3: 制御系設計

◆電流制御ループ
1. インダクタンスLから電流制御対象を導出
2. 電流ループ帯域を fs/10 に設定
3. PI制御器パラメータを計算

◆電圧制御ループ  
1. 出力キャパシタンスCと電流ループから電圧制御対象を導出
2. 電圧ループ帯域を電流ループ帯域の1/10に設定
3. PI制御器パラメータを計算

5.4 Step 4: 安定性解析
- ボード線図による安定余裕確認
- 位相余裕: 45度以上
- ゲイン余裕: 6dB以上

6. EMC・保護機能の考慮事項
================================================================================

6.1 EMC（電磁両立性）設計

◆ノイズ発生源
- スイッチング動作: dv/dt, di/dtによる放射ノイズ
- 整流ダイオード: 逆回復電流によるノイズ
- 制御回路: デジタル信号によるクロックノイズ

◆EMC対策手法
1. 回路設計:
   - スナバ回路の追加
   - ソフトスイッチング技術
   - 適切なレイアウト設計

2. フィルタ設計:
   - 入力EMIフィルタ
   - 出力フィルタ
   - コモンモードチョーク

6.2 保護機能

◆過電流保護
- 検出方法: シャント抵抗、電流トランス
- 保護動作: 即座停止、ソフト電流制限

◆過電圧保護
- 入力過電圧: ツェナーダイオード、サージアブソーバ
- 出力過電圧: 電圧監視回路、クラウバ回路

◆過熱保護
- 温度検出: サーミスタ、温度スイッチ
- 保護動作: ディレーティング、シャットダウン

================================================================================
設計例：24V→12V Buckコンバータ
================================================================================

◆仕様
- 入力電圧: 24V ± 10%
- 出力電圧: 12V
- 最大出力電流: 5A
- スイッチング周波数: 100kHz
- 出力電圧リップル: < 1%

◆設計計算
デューティ比: D = Vout/Vin = 12/24 = 0.5

インダクタ設計:
ΔI_L = 0.3 × 5A = 1.5A
L = (24-12) × 0.5 / (1.5 × 100k) = 40μH

出力キャパシタ設計:
ΔVout = 12V × 0.01 = 0.12V
C = 1.5 / (8 × 100k × 0.12) = 156μF

電流制御ループ設計:
電流ループ帯域: ω_ci = 2π × 10kHz
Ti_i = L/R = 40μH/負荷抵抗 (極零相殺)
Kp_i = L × ω_ci = 40μH × 2π × 10k

電圧制御ループ設計:  
電圧ループ帯域: ω_cv = 2π × 1kHz
Ti_v = 1/ω_cv = 1/(2π × 1k)
Kp_v = C × ω_cv² = 156μF × (2π × 1k)²
