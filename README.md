## 🎯 概要

**状態空間モデルに基づく厳密な理論**により、DC-DC昇圧コンバーターの制御システムを自動設計するPythonツールです。内外二重PI制御（電流ループ+電圧ループ）の最適化を行い、安定余裕を保証した設計を提供します。

### 主要機能

- ✅ **厳密な状態空間モデリング**: 平均化モデルによる正確な系表現
- ✅ **伝達関数自動導出**: 制御-出力、ライン-出力特性の数学的解析
- ✅ **内外二重PI制御設計**: 電流ループと電圧ループの最適パラメータ算出
- ✅ **安定性解析**: 位相余裕・ゲイン余裕の定量評価
- ✅ **RHP零点制約考慮**: 昇圧コンバーター特有の右半面零点制約
- ✅ **Excel/HTML出力**: 詳細な設計レポートと技術文書生成

## 📊 設計プロセス

### 1. 動作点解析
```python
# デューティ比計算
D = (Vo - Vin) / Vo = (12 - 8) / 12 = 0.333

# 出力電流
Io = Vo / R

# 平均インダクタ電流
IL_avg = Io / (1 - D)

# CCM条件チェック
Lmin = (Vin * D * (1-D)² * R) / (2 * fsw * Vo)
```

### 2. 状態空間モデル
```python
# 状態変数: x = [iL, vC]^T
# 状態方程式: dx/dt = A*x + B*u

A = [[0, -(1-D)/L],
     [(1-D)/C, -1/(RC)]]

B_control = [[Vo/L],
             [-Io/C]]
```

### 3. 伝達関数導出
```python
# 制御-出力電圧伝達関数
Gvd(s) = (Vin/(1-D)²) * (1 + s*Rz*C) / (1 + s/ωp + s²/ωp²)

# 右半面零点
f_RHP = (1-D)²*R / (2π*D²*L*C)
```

### 4. PI制御器設計
```python
# 電流ループ設計
f_ci = fsw / 10                    # クロスオーバー周波数
Kp_i = 1 / |Gid(jω_ci)|           # 比例ゲイン
Ki_i = Kp_i / Ti_i                 # 積分ゲイン

# 電圧ループ設計
f_cv = f_ci / 7                    # 電流ループより低周波
f_cv < f_RHP / 5                   # RHP零点制約
```
