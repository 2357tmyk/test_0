#### 1. 昇圧コンバーターの状態空間モデリング
**学習内容:**

- 二状態（ON/OFF）からの連続時間近似

**重要な知見:**
```python
# 状態空間モデル: dx/dt = A*x + B*u

```
- デューティ比Dが系の極配置に直接影響
- 負荷変動時の動的特性予測が可能

#### 2. 右半面零点(RHP Zero)の理解
**学習内容:**
- 昇圧コンバーター特有の非最小位相特性
- f_RHP = (1-D)²R/(2πD²LC) の物理的意味
- 制御帯域幅制限の根本原因

**設計への影響:**
- 電圧ループ帯域: f_cv < f_RHP/5 (安全率考慮)
- 応答性と安定性のトレードオフ関係

#### 3. 内外二重制御の設計思想
**学習内容:**
- 電流ループ(内側): 高速応答、電流制限機能
- 電圧ループ(外側): 精密制御、負荷変動対応
- 周波数分離の重要性: f_ci ≥ 5×f_cv

**実装上のポイント:**
```python
f_ci = fsw / 10        # 電流ループ帯域
f_cv = f_ci / 7        # 電圧ループ帯域 (分離確保)
```
