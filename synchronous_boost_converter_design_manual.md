# 同期整流方式昇圧回路 標準設計手順書
## ～サージ・EMC・熱設計（損失の厳密分離）を包含したパワー段実務編～

### 概要
本手順書は、同期整流方式昇圧DCDCコンバータにおけるパワー段コンポーネントの選定、基板寄生要素に起因するサージ・EMC対策、および熱設計を網羅した実務用の設計手順である。多様な電気設計技術者（ハード設計・熱設計・EMC・半導体・信頼性確保エンジニア）の視点から体系的に検証・改善を行った汎用的な設計ガイドラインを提供する。

---

## 0. 昇圧回路 基本理論式

### 0.1 理想昇圧回路の基本関係式
- **出力電圧**  
  `V_out = V_in / (1 - D)`
  
- **インダクタ電流リプル**  
  `ΔI_L = V_in × D / (L × f_s)`
  
- **出力電圧リプル（容量性）**  
  `ΔV_out_C = I_out × D / (C × f_s)`
  
- **出力電圧リプル（ESR性）**  
  `ΔV_out_ESR = ΔI_L × ESR`

### 0.2 動作モード判定条件
- **CCM/DCM境界電流**  
  `I_boundary = (V_in × D × (1-D)^2) / (2 × L × f_s)`
  
- **動作モード判定**  
  - `I_out_min > I_boundary` → CCM（連続導通モード）
  - `I_out_min < I_boundary` → DCM（不連続導通モード）

## 1. 設計仕様の定義とパラメータ入力

### 1.1 基本仕様パラメータ
- **V_in** : 入力電圧（変動範囲：V_in(min) ～ V_in(max)）[V]
- **V_out** : 目標出力電圧 [V]
- **I_out(max)** : 最大出力負荷電流 [A]
- **f_sw** : スイッチング周波数（周期：T_sw = 1 / f_sw）[Hz]
- **r** : インダクタ電流リプル比（推奨値：0.2 ～ 0.4）[-]
- **ΔV_out(max)** : 許容最大出力電圧リプル [V]
- **η** : 想定効率（初期計算用、通常 0.85 ～ 0.95 と仮定）[-]

### 1.2 環境・信頼性パラメータ
- **T_a(max)** : 機器の想定最大周囲温度（熱設計の基準環境温度）[°C]
- **T_j(max)** : 半導体接合部最大許容温度 [°C]（通常120°C以下で設計）
- **MTBF_target** : 目標平均故障間隔 [hours]
- **Safety_factor** : 安全係数（通常1.5～2.0）[-]

### 1.3 高周波・熱設計関連の寄生パラメータ
- **L_loop** : 高周波ホットループの寄生インダクタンス [nH]
- **C_sw** : スイッチング（SW）ノードの全寄生容量 [pF]
- **V_spike(max)** : 許容最大サージ電圧 [V]
- **R_th(ja)** : 各部品の接合部－周囲間熱抵抗（基板放熱を含めた実効値）[°C/W]

## 2. 実効デューティサイクルの厳密算定

### 2.1 理想デューティサイクルの計算
理想的な昇圧回路における基本デューティサイクル：
```
D_ideal = 1 - V_in / V_out
```

### 2.2 実効デューティサイクル（CCM時）
デッドタイム（T_d）およびパワー段の寄生抵抗成分を考慮した実効最大デューティサイクル：

```
D_ctrl(max) = 1 - [V_in(min) - I_in(max) × (R_on_LS × D + DCR) - V_F × 2 × f_sw × T_d] / V_out
```

ここで：
- **R_on_LS** : ローサイドMOSFETのオン抵抗 [Ω]
- **DCR** : インダクタ直流抵抗 [Ω]
- **V_F** : MOSFETボディダイオード順電圧 [V]
- **T_d** : デッドタイム [s]

### 2.3 デッドタイム最適化指針
```
T_d(opt) = √(Q_rr × L_loop / I_in(max))
```
ただし、以下の制約条件を満たすこと：
- **最小値**: `T_d > 10 ns`（ドライバIC特性による下限）
- **最大値**: `T_d < T_sw / 20`（効率低下防止）

### 2.4 超軽負荷時の電圧上昇特性
負荷が極めて軽い場合の出力電圧上昇：
```
V_out(light_load) = [V_in + 2 × f_sw × T_d × V_F] / [1 - D_ctrl - 2 × f_sw × T_d]
```

## 3. 主インダクタ（L）の選定と検証手順

### 3.1 インダクタンス値の決定
最大入力電流から必要インダクタンスを算出：

```
I_in(max) = (V_out × I_out(max)) / (V_in(min) × η)
ΔI_L = I_in(max) × r
L_min = (V_in(min) × D_ctrl(max)) / (ΔI_L × f_sw)
```

実際の選定値：`L_actual = L_min × 1.2`（20%マージン）

### 3.2 飽和特性の検証
```
I_L(peak) = I_in(max) + (ΔI_L / 2)
I_sat(required) > I_L(peak) × 1.3   （30%マージン）
```

### 3.3 インダクタ発熱の詳細解析
実効値電流とインダクタ損失：
```
I_L(rms) = √[I_in(max)^2 + (ΔI_L^2 / 12)]
P_L_copper = I_L(rms)^2 × DCR    （銅損）
P_L_core = K × f_sw^α × B_peak^β （鉄損、材料依存）
P_L_total = P_L_copper + P_L_core
```

インダクタ温度上昇：
```
T_L = T_a(max) + (P_L_total × R_th_L)
```
判定基準：`T_L < T_L(max)`（通常125°C～150°C）

### 3.4 EMC観点での検証事項
- **自己共振周波数（SRF）**: `SRF > 20 × f_sw`
- **Q値**: 高周波特性とEMI抑制のバランス
- **シールド構造**: 放射ノイズ抑制効果

## 4. MOSFET詳細損失解析（熱設計の核心）

### 4.1 ローサイドMOSFET（Q_LS）の総損失
ローサイドMOSFETは同期整流回路で最も過酷な熱負荷を受ける：

```
P_LS = P_cond_ls + P_sw_ls + P_Qrr + P_Coss + P_gate_ls
```

#### 4.1.1 導通損失
```
P_cond_ls = I_in(max)^2 × R_on_LS × D_ctrl(max)
```

#### 4.1.2 スイッチング損失（純粋分）
```
P_sw_ls = 0.5 × V_out × I_in(max) × (t_r + t_f) × f_sw
```
注：t_r, t_fはゲート抵抗R_g(on), R_g(off)により調整可能

#### 4.1.3 逆回復損失（ハイサイド起因）
```
P_Qrr = V_out × Q_rr × f_sw
```
注：Q_HSのボディダイオード逆回復電荷がQ_LSを通して流れるため

#### 4.1.4 出力容量充放電損失
```
P_Coss = 0.5 × (C_oss_LS + C_oss_HS) × V_out^2 × f_sw
```

#### 4.1.5 ゲート駆動損失
```
P_gate_ls = Q_g_ls × V_cc × f_sw
```

### 4.2 ハイサイドMOSFET（Q_HS）の総損失
ハイサイドはZVSによりスイッチング損失が大幅に軽減：

```
P_HS = P_cond_hs + P_dead_hs + P_gate_hs
```

#### 4.2.1 導通損失
```
P_cond_hs = I_in(max)^2 × R_on_HS × (1 - D_ctrl(max))
```

#### 4.2.2 デッドタイム中のダイオード損失
```
P_dead_hs = V_F × I_in(max) × 2 × T_d × f_sw
```

#### 4.2.3 ゲート駆動損失
```
P_gate_hs = Q_g_hs × V_cc × f_sw
```

## 5. 半導体の熱設計と信頼性評価

### 5.1 ジャンクション温度計算
```
T_j_LS = T_a(max) + (P_LS × R_th(ja)_LS)
T_j_HS = T_a(max) + (P_HS × R_th(ja)_HS)
```

### 5.2 ディレーティング設計基準
- **絶対最大定格**: 150°C（一般的なシリコンMOSFET）
- **設計目標温度**: T_j < 120°C（定格の80%）
- **車載グレード**: T_j < 125°C

### 5.3 信頼性予測（アレニウスモデル）
```
AF = exp[Ea/k × (1/T_j1 - 1/T_j2)]
MTBF_actual = MTBF_base × AF
```
ここで：
- **Ea** : 活性化エネルギー（0.7 eV）
- **k** : ボルツマン定数
- **T_j1, T_j2** : 絶対温度 [K]

### 5.4 熱対策アルゴリズム
1. **T_j_LS過大時**: 
   - Q_HSの選定見直し（低Q_rr、低C_ossを優先）
   - サーマルビア増設によるR_th(ja)低減
2. **T_j_HS過大時**:
   - R_on_HS低減またはパッケージ大型化
   - デッドタイム最適化

## 6. サージ抑制設計と損失の統合評価

### 6.1 RCスナバ回路設計
SW-GND間にRCスナバを配置し、サージ・EMI対策を実施：

#### 6.1.1 寄生パラメータの実測・推定
共振周波数測定から逆算：
```
C_sw = C_add / 3    （実験的経験値）
L_loop = 1 / [(2π × f_ring1)^2 × C_sw]
```

#### 6.1.2 最適スナバ定数
```
C_snub = 3 × C_sw
R_snub = √(L_loop / C_sw)
```

### 6.2 スナバ抵抗の損失と熱設計
```
P_R_snub = C_snub × V_spike^2 × f_sw
```
選定基準：定格電力 > 2 × P_R_snub（パルス耐性考慮）

### 6.3 包括的サージ対策
1. **レイアウト最適化**: L_loop最小化
2. **TVSダイオード**: 半導体保護
3. **フェライトビーズ**: 高周波ノイズ抑制
4. **差動フィルタ**: コモンモード・ノーマルモード対策

## 7. 出力コンデンサの選定とリプル熱設計

### 7.1 容量値の決定
同期整流昇圧では、D期間中に負荷電流すべてがコンデンサから供給される：

```
ΔV_out_C = (I_out(max) × D_ctrl(max)) / (C × f_sw)
ΔV_out_ESR = [I_in(max) + (ΔI_L / 2)] × ESR
ΔV_out_total = ΔV_out_C + ΔV_out_ESR
```

DCバイアス特性を考慮した実効容量で評価すること。

### 7.2 リプル電流による発熱
出力コンデンサに流入する実効値電流：
```
I_C_out(rms) = I_out(max) × √[D_ctrl(max) × (1 - D_ctrl(max))]
```

コンデンサ自己発熱：
```
P_C_out = I_C_out(rms)^2 × ESR
```

### 7.3 コンデンサ選定基準
- **リプル電流定格**: `I_ripple_rating > I_C_out(rms) × 1.5`
- **耐電圧**: `V_rating > V_out × 1.3`
- **温度特性**: 使用温度範囲での容量変化率考慮
- **ESL特性**: 高周波インピーダンス低減

## 8. 基板レイアウト設計指針（熱・EMC統合設計）

### 8.1 高周波ホットループの最適化
**最重要原則**: Q_HS - Q_LS - C_out(HF) の3点間距離を極限まで短縮

具体的指針：
- 部品間距離：< 5mm（理想的には< 2mm）
- 配線幅：可能な限り太く（> 1mm）
- 層間接続：最短経路でビア配置

### 8.2 グランドプレーン設計
- **パワーGND**: 分断のない連続プレーン
- **信号GND**: パワーGNDと1点接続
- **リターンパス**: 信号線直下にGNDプレーン配置

### 8.3 熱設計レイアウト
#### 8.3.1 熱源分散配置
主要発熱部品（Q_LS, Q_HS, L）の相互干渉回避：
- 最小離間距離：> 10mm
- 熱勾配を考慮した配置方向性

#### 8.3.2 サーマルマネジメント
- **サーマルビア**: 発熱部品直下に多数配置（φ0.2mm, ピッチ0.5mm）
- **放熱パッド**: 最大面積確保
- **エアフロー**: 自然対流または強制冷却を考慮した部品配置

### 8.4 EMC対策レイアウト
#### 8.4.1 ノイズ源の封じ込め
- **差動信号**: ツイストペア配線またはストリップライン
- **クロック信号**: シールド配線とガードトレース
- **SW ノード**: 最小面積・最短配線

#### 8.4.2 フィルタ配置戦略
- **入力フィルタ**: 電源入力直後に配置
- **出力フィルタ**: 負荷直前に配置
- **局所バイパス**: 各IC電源ピン近傍

## 9. 設計検証・最適化プロセス

### 9.1 初期設計検証項目
1. **基本動作**: CCM/DCM動作点確認
2. **効率特性**: 負荷変動時の効率測定
3. **熱特性**: 定常温度分布測定
4. **EMC特性**: 伝導・放射ノイズ測定

### 9.2 最適化サイクル
#### Phase 1: 性能最適化
- スイッチング周波数調整
- デッドタイム最適化
- ゲート抵抗調整

#### Phase 2: 信頼性最適化  
- ストレス解析
- 加速劣化試験
- 故障モード解析（FMEA）

#### Phase 3: コスト最適化
- 部品統合・削減
- 実装面積最小化
- 歩留まり向上

### 9.3 設計完了判定基準
すべての項目が基準値内であることを確認：
- **効率**: η > 目標値
- **温度**: T_j < 設計制限値
- **EMC**: 規格適合
- **信頼性**: MTBF > 目標値
- **コスト**: 目標コスト内

## 10. 実装上の注意事項とトラブルシューティング

### 10.1 よくある設計問題と対策
#### 問題1: ローサイドMOSFET過熱
**原因**: ハイサイドのQ_rr・C_ossが大きい
**対策**: ハイサイド素子の見直し、スナバ回路最適化

#### 問題2: EMIノイズ過大
**原因**: ホットループ面積が大きい、リターンパス不適切
**対策**: レイアウト見直し、フィルタ強化

#### 問題3: 出力電圧リプル過大
**原因**: 出力容量不足、ESR過大
**対策**: 容量値増加、低ESRコンデンサ並列接続

### 10.2 設計マージン設定指針
- **電流定格**: 1.5倍以上
- **電圧定格**: 1.3倍以上  
- **電力定格**: 2.0倍以上
- **温度定格**: 定格の80%以下で使用
